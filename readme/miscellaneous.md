# Miscellaneous

Listing contents of .vdh file:

```
7z l <filename.vdh>
```

Mounting a .vhd file:

<pre><code><strong>sudo guestmount --add &#x3C;filname.vdh> --inspector --ro -v /mnt/vhd
</strong></code></pre>

To get image metadata:

```
exiftool name.jpg
```

Cracking a zip file:

```
fcrackzip -D -p wordlist file.zip
```
